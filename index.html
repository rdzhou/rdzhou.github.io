How to Disable Show More Options in Context Menu of Windows 11

Launch Command Prompt to run the following commands to disable or enable.

  • Disable:
    • reg add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve
  • Enable:
    • reg delete "HKEY_CURRENT_USER\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}" /f​

How to Quickly Open Specific Control Panel Options

  • Press Win + R to open Run
  • Enter xxx.cpl and Press enter

Example Options:

  • Network Connections: ncpa.cpl
  • Region (convenient for setting language for non-Unicode programs): intl.cpl
  • Sound(Multimedia properties): mmsys.cpl

tmux with mosh

  • Terminal emulator supporting OSC52
    • iTerm2, Preference -> General -> Applications in terminal may access clipboard
    • mlterm, –osc52=true
  • latest mosh from git (tagged releases are too old, both client and server)
  • latest tmux
  • Override the OSC 52 sequence generated by tmux by adding the following line to your tmux.conf
    • set-option -ag terminal-overrides ",xterm-256color:Ms=\\E]52;c;%p2%s\\7"

Reference
